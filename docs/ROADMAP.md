# Roadmap MiTienda 2.0

## Sprint 1 - Core Ecommerce

Objetivo:

Levantar MedusaJS y PostgreSQL.

Entregables:

* Proyecto MedusaJS
* PostgreSQL
* Productos
* Categorías
* Inventario
* Clientes
* Carrito
* Checkout
* Órdenes
* Panel administrativo

Resultado:

Backend ecommerce funcional.

---

## Sprint 2 - Modelo de Datos MiTienda

Objetivo:

Diseñar entidades personalizadas.

Entregables:

* Brand
* Vendor
* Shipping Rules
* Extensiones de Product

Resultado:

Modelo de datos preparado para marketplace.

---

## Sprint 3 - Marketplace

Objetivo:

Implementar vendedores.

Entregables:

* Gestión de emprendedores
* Productos por vendedor
* Perfil público de vendedor
* Roles y permisos

Resultado:

Marketplace básico operativo.

## Sprint 4 - Promotion Engine

Objetivo:

Motor comercial flexible.
